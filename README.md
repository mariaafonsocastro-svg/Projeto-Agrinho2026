# Maria Clara Website

## 📖 Sobre o Projeto

Este projeto consiste em um website desenvolvido em HTML, CSS e JavaScript com o objetivo de apresentar informações de forma moderna, responsiva e interativa.

## 🚀 Funcionalidades

* Interface responsiva para dispositivos móveis e desktops;
* Navegação intuitiva;
* Elementos visuais modernos;
* Animações e efeitos interativos;
* Conteúdo organizado em seções;
* Compatibilidade com os principais navegadores.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts

## 📂 Estrutura do Projeto

```text
/
├── maria.html
├── css/
├── js/
├── images/
└── README.md
```

## ▶️ Como Executar

1. Faça o download dos arquivos do projeto.
2. Abra o arquivo `maria.html` em qualquer navegador moderno.
3. Navegue pelo site normalmente.

## 🎨 Personalização

Você pode modificar:

* Cores no arquivo CSS;
* Textos diretamente no HTML;
* Funcionalidades no JavaScript;
* Imagens e ícones conforme necessário.

## 📱 Responsividade

O site foi projetado para funcionar em:

* Computadores
* Tablets
* Smartphones
